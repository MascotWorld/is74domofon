# IS74 Domofon Add-on

Home Assistant Add-on для запуска API сервера IS74 Домофон.

## Установка

### Способ 1: Локальный add-on

1. Скопируйте папку `is74-domofon-addon` в `/addons/` на вашем Home Assistant:
   ```bash
   scp -r is74-domofon-addon root@homeassistant.local:/addons/is74_domofon
   ```

2. Перейдите в **Настройки → Дополнения → Магазин дополнений**

3. В правом верхнем углу нажмите **⋮ → Проверить наличие обновлений**

4. Найдите "IS74 Domofon" в разделе "Локальные дополнения"

5. Установите и запустите

### Способ 2: Через репозиторий

1. Перейдите в **Настройки → Дополнения → Магазин дополнений**

2. Нажмите **⋮ → Репозитории**

3. Добавьте URL: `https://github.com/your-username/is74domofon`

4. Найдите и установите "IS74 Domofon"

## Настройка

После установки add-on:

1. Откройте конфигурацию add-on
2. Установите уровень логирования (по умолчанию: info)
3. Запустите add-on
4. Откройте веб-интерфейс через боковую панель

## Первоначальная авторизация

1. Откройте веб-интерфейс add-on
2. Введите номер телефона и запросите код
3. Введите код из SMS
4. После успешной авторизации перезапустите add-on

## Конфигурация

Файлы конфигурации хранятся в `/config/is74_domofon/`:

- `tokens.json` - токены авторизации
- `fcm_creds.json` - данные FCM для push-уведомлений

## Использование с интеграцией

В настройках интеграции IS74 Domofon используйте URL:

- Если add-on и HA на одном хосте: `http://localhost:10777`
- Через Ingress: используйте URL из боковой панели
